# Proj

**Proj** — A fast and extensible CLI tool for scaffolding Python, Web, and multi‑language projects using template‑based configs.

![rust](https://img.shields.io/badge/language-Rust‑%23791‑color)
![license](https://img.shields.io/badge/license‑MIT‑blue)

## 🚀 About

**Proj** is a lightweight CLI that accelerates project setup by generating a project structure from reusable templates.

If you find yourself repeatedly creating boilerplate code for new projects (Python packages, web apps, microservices, etc.), then **Proj** helps streamline that with customizable templates.

## 📦 Features

- 🛠️ **CLI driven** — Generate a new project from the command line
- 📄 **Template‑based** — Use configurable templates for structure and starter files
- 🌐 Language‑agnostic support — Not limited to one language or framework
- ⚡ Extensible — Add your own templates easily
